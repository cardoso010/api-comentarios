# api-comentarios

Projeto desenvolvido para a realização de um desafio.

### Prerequisites

```
PHP7
COMPOSER
MYSQL
APACHE2 OR NGINX
```

## Getting Started

```
git clone https://github.com/cardoso010/api-comentarios
cd api-comentarios/
sudo composer update
```
